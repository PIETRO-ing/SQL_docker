# SQL & docker

![](./Images/psql.png) 
![](./Images/docker.png)

The main purpose of this project is to practice docker-compose and sql.

## 1st step
Built a database called 'company' which consists of three tables:
- departments;
- regions;
- employees.
The database is dockerized and created by using sqlalchemy.

## 2nd step 
Practicing queries:
- aggregate functions;
- subqueries;
- case;
- join table.
